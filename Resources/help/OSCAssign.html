OSC (Open Sound Control) Assignment


Configuring a slider to respond to an OSC message is very easy.


1) To open the configuration window, double-click on a slider's label.



2) Set the incoming port and address in the format port:address The message must be a single floating-point value between 0 and 1. The value is automatically mapped to the slider's boundaries.


The second field allows to set an outgoing message that can be used to initialize the OSC controller with the current value of the slider when the performance is launched. The format is host:port:address where host is the IP address of the controller.


To configure a range slider to respond to two OSC messages, first, double-click on the left half of the label to set the address of the left handle and then, double-click on the right half of the label to set the address of the right handle.
